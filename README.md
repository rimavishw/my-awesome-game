# 🐍 Snake Gesture Game

A modern twist on the classic Nokia Snake — controlled by hand gestures via webcam!

## ✨ Features
- 🖐️ Real-time hand tracking via MediaPipe Hands
- 📷 Live webcam — you appear inside the game!
- 🌟 Neon snake with particle effects
- 🔊 Sound effects & score tracking
- 🫲 Open palm to restart

## 🛠️ Built With
- HTML, CSS, Vanilla JavaScript
- Canvas API
- MediaPipe Hands

## 🚀 How to Play
1. Allow webcam access
2. Move your hand to control the snake
3. Open palm gesture to restart

## 💡 About
Built using vibe coding — pure web tech, no frameworks!
